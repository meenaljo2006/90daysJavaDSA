### Day-1 Initial Learning Phase : Java Fundamentals
During the initial phase of this challenge, the focus was on revising core programming and Java fundamentals to build a strong foundation before moving into Data Structures and Algorithms.

📚 This included revisiting programming basics such as the Introduction to programming languages, different programming paradigms (procedural, functional, and object-oriented), and the distinction between static and dynamic languages. Alongside this, I revised essential Java concepts including Java’s introduction, code execution process, and architecture. I also worked on problem-solving foundations through flowcharts and pseudocode, followed by a structured review of Java basics such as variables and data types, operators, conditional statements, loops (flow control), and functions/methods.
